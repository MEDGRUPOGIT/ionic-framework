# Label

Documentaçao complementar ion-label: **https://ionicframework.com/docs/api/label**

## Propriedades

| Propriedade | Atributo    | Descrição                                     | Tipo                          | Default       |
| ----------- | ----------- | --------------------------------------------- | ----------------------------- | ------------- |
| **dsName**  | **ds-name** | Define o estilo estilo do label dentro do DS. | **"stacked"** \| **"helper"** | **undefined** |

## Usage

### stacked

Usada para complementar o componente **ion-input** e adicionar o estilo stacked.

```html
  <ion-label ds-name="stacked">Stacked</ion-label>
  <ion-input placeholder="Enter Input"></ion-input>
```

### helper

Usada para complementar o componente **ion-input** e adicionar o estilo helper.

```html
  <ion-input placeholder="Enter Input"></ion-input>
  <ion-label ds-name="helper">Helper</ion-label>
```
