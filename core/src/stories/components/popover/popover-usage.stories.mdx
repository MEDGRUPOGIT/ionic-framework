# Popover

Documentação auxiliar popover: **https://ionicframework.com/docs/api/popover**

**As propriedades marcadas com ⚠️ são extritamente necessárias.**

```ts
async presentPopover(ev: any) {
  const popover = await this.popoverController.create({
    cssClass: 'med-popover', ⚠️
    mode: 'ios', ⚠️
    showBackdrop: true ⚠️
  });
  await popover.present();
}
```
