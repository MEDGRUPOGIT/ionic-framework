# Accordion

## Propriedades

| Propriedade   | Atributo       | Descrição                      | Tipo                      | Default       |
| ------------- | -------------- | ------------------------------ | ------------------------- | ------------- |
| **size**      | **size**       | Aplica tamanho fullscreen.     | **full**                  | **undefined** |
| **icon**      | **icon**       | Define a posição do ícone.     | **"left"** \| **"right"** | **undefined** |
| **collapsed** | **collapsed**  | Define o estado do componente. | **boolean**               | **true**      |

## Métodos

### `toggle() => Promise<void>`

## Usage

```html
<med-accordion></med-accordion>

<!-- size -->
<med-accordion size="full"></med-accordion>

<!-- icon -->
<med-accordion icon="left"></med-accordion>
<med-accordion icon="right"></med-accordion>

<!-- collapsed -->
<med-accordion collapsed></med-accordion>
```
