# Accordion

## Propriedades

| Propriedade  | Atributo  | Descrição                   | Tipo    | Default       |                                                                                            | Default       |
| ------------ | --------- | --------------------------- | ------- | ------------- |
| **size**     | **size**  | Aplica tamanho fullscreen.  | string  | **undefined** |

## Usage

```html
<med-accordion></med-accordion>

<!-- size -->
<med-accordion size="full"></med-accordion>
```
