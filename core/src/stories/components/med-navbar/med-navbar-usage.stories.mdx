# Navbar

## default

### Slots

| Slot                | Descrição                                        |
| ------------------- | ------------------------------------------------ |
| **slot="left"**     | Posiciona um elemento no começo da navbar.       |
| **slot="title"**    | Titulo da navbar.                                |
| **slot="subtitle"** | Subtitulo da navbar.                             |
| **slot="right"**    | Posiciona um elemento no lado direito da navbar. |

```html
<med-navbar>
  <ion-button ds-name="icon-label" slot="left">
    <ion-icon name="med-chevron-left"></ion-icon>
    voltar
  </ion-button>

  <span slot="title">Título</span>
  <span slot="subtitle">subtítulo</span>

  <ion-button ds-name="icon-only" slot="right">
    <ion-icon slot="icon-only" name="med-star-filled"></ion-icon>
  </ion-button>
</med-navbar>
```

## transparent

### Propriedades

| Propriedade  | Atributo     | Descrição                | Tipo             | Default         |
| ------------ | ------------ | ------------------------ | ---------------- | --------------- |
| **dsName**   | **ds-name**  | Define a navbar.         | **"secondary"**  | **"secondary"** |
| **dsTheme**  | **ds-theme** | Define o tema da navbar. | **"secondary"**  | **"secondary"** |


### Atributos

| slot                | Descrição                                                              | valores            |
| ------------------- | ---------------------------------------------------------------------- | ------------------ |
| **ds-name**         | Define a navbar.                                                       | **transparent**    |
| **ds-theme**        | Define a cor dos itens da navbar em contrate a determinado background. | **dark**/**light** |

### Slots

| Slot                | Descrição                                        |
| ------------------- | ------------------------------------------------ |
| **slot="left"**     | Posiciona um elemento no começo da navbar.       |
| **slot="title"**    | Titulo da navbar.                                |
| **slot="subtitle"** | Subtitulo da navbar.                             |
| **slot="right"**    | Posiciona um elemento no lado direito da navbar. |

```html
<med-navbar ds-name="transparent" ds-theme="dark">
  <ion-button ds-name="icon-label" slot="left">
    <ion-icon name="med-chevron-left"></ion-icon>
    voltar
  </ion-button>

  <span slot="title">Título</span>
  <span slot="subtitle">subtítulo</span>

  <ion-button ds-name="icon-only" slot="right">
    <ion-icon slot="icon-only" name="med-star-filled"></ion-icon>
  </ion-button>
</med-navbar>
```
