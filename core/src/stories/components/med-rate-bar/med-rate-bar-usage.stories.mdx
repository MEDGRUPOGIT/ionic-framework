# Rate Bar

**Este componente funciona como wrapper para o componente [med-rate-like](http://desenv.ordomederi.com/med-components/?path=/story/components-global-rate-like--default)**.

```html
<!-- default -->
<med-rate-bar>
  Avalie esse vídeo
  <med-rate-like></med-rate-like>
</med-rate-bar>

<!-- status -->
<med-rate-bar>
  Avalie esse vídeo
  <med-rate-like status="like"></med-rate-like>
</med-rate-bar>
```
