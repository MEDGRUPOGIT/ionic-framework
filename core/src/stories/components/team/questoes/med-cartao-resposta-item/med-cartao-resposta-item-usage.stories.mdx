# Usage

### Propriedades

| Propriedade                  | Atributo     | Descrição                                                                    | Tipo                                                                                                                   | Default       |
| ---------------------------- | ------------ | ---------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**                  | **ds-color** | Define a cor do componente.                                                  | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**  | **undefined** |
| **dsName**                   | **ds-name**  | Define a variação do componente.                                             | **"secondary"**  \| **undefined**                                                                                      | **undefined** |
| **dsSize**                   | **ds-size**  | Define a variação de tamanho do componente.                                  | **"xxs"** \| **"xs"** \| **"sm"** \| **"md"** \| **"lg"** \| **undefined**                                             | **undefined** |
| **color="success-medium"**   |              | Este atributo deve ser utilizado quando a questão estiver **correta**.       | **success-medium**                                                                                                     | **undefined** |
| **color="error-medium"**     |              | Este atributo deve ser utilizado quando a questão estiver **errada**.        | **error-medium**                                                                                                       | **undefined** |
| **color="warning-medium"**   |              | Este atributo deve ser utilizado quando a questão estiver **anulada**.       | **warning-medium**                                                                                                     | **undefined** |

```
<med-cartao-resposta-lista>
  <!-- default -->
  <med-cartao-resposta-item>01</med-cartao-resposta-item>

  <!-- ativo -->
  <med-cartao-resposta-item ativa>02</med-cartao-resposta-item>

  <!-- correto -->
  <med-cartao-resposta-item dsColor="fb-success">03</med-cartao-resposta-item>

  <!-- incorreto -->
  <med-cartao-resposta-item dsColor="fb-caution">04</med-cartao-resposta-item>

  <!-- anulada -->
  <med-cartao-resposta-item dsColor="fb-warning" anulada>05</med-cartao-resposta-item>

  <!-- impressa -->
  <med-cartao-resposta-item impressa>06</med-cartao-resposta-item>
</med-cartao-resposta-lista>
```
