# Usage

### Atributos

| Event                        | Descrição                                                                    | Valores                |
| ---------------------------- | ---------------------------------------------------------------------------- | ---------------------- |
| **color="success-medium"**   | Este atributo deve ser utilizado quando a questão estiver **correta**.       | **success-medium**     |
| **color="error-medium"**     | Este atributo deve ser utilizado quando a questão estiver **errada**.        | **error-medium**       |
| **color="warning-medium"**   | Este atributo deve ser utilizado quando a questão estiver **anulada**.       | **warning-medium**     |
| **active**                   | Este atributo deve ser utilizado quando a questão for a ativa (selecionada). | **Atributo sem valor** |
| **anulada**                  | Este atributo deve ser utilizado quando a questao for anulada.               | **Atributo sem valor** |
| **impressa**                 | Este item é um slot para quando a questaofor impressa                        | **Atributo sem valor** |

```
<med-cartao-resposta-lista>
  <!-- default -->
  <med-cartao-resposta-item>01</med-cartao-resposta-item>

  <!-- ativo -->
  <med-cartao-resposta-item active>02</med-cartao-resposta-item>

  <!-- correto -->
  <med-cartao-resposta-item color="success-medium">03</med-cartao-resposta-item>

  <!-- incorreto -->
  <med-cartao-resposta-item color="error-medium">04</med-cartao-resposta-item>

  <!-- anulada -->
  <med-cartao-resposta-item color="warning-medium" anulada>05</med-cartao-resposta-item>

  <!-- impressa -->
  <med-cartao-resposta-item impressa>06</med-cartao-resposta-item>
</med-cartao-resposta-lista>
```
