# Progress Bar

## Propriedades

| Propriedade    | Atributo       | Descrição                          | Tipo             | Default          |
| -------------- | -------------- | ---------------------------------- | ---------------- | ---------------- |
| **value**      | **value**      | Define o botão.                    | **"icon-label"** | **"icon-label"** |
| **percentage** | **percentage** | Define o estado disabled do botão. | **boolean**      | **undefined**    |

| Atributo       | Descrição                              | Type        | Valor       |
| -------------- | -------------------------------------- | ----------- | ----------- |
| **value**      | Define o valor para progresso da barra | **Number**  | > 0  <= 0.1 |
| **percentage** | Esconde ou mostra a porcentagem        | **Boolean** |             |

```html
<!-- value -->
<ion-progress-bar value="0.3"></ion-progress-bar>

<!-- percentage -->
<ion-progress-bar value="0.3" percentage="true"></ion-progress-bar>
```
