# Rate Like

## Propriedades

| Propriedade  | Atributo   | Descrição             | Tipo           | Default       |
| ------------ | ---------- | --------------------- | -------------- | ------------- |
| **status**   | **status** | Define o botão ativo. | **RateStatus** | **undefined** |

## Eventos

| Event         | Descrição                                                   | Tipo            | Default       |
| ------------- | ----------------------------------------------------------- | --------------- | ------------- |
| **medChange** | Emitido quando é feito um click no botão de like ou dislike | **RateStatus**  | **undefined** |

```html
<!-- default -->
<med-rate-like></med-rate-like>

<!-- status -->
<med-rate-like status="like"></med-rate-like>
```

```ts
enum RateStatus {
  LIKE = 'like',
  DISLIKE = 'dislike'
}
```
