# Banner

## Propriedades

| Propriedade               | Atributo      | Descrição                                          | Tipo       | Default       |
| ------------------------- | ------------- | -------------------------------------------------- | ---------- | ------------- |
| **header** _(required)_   | **header**    | Define o título do banner.                         | **string** | **undefined** |
| **btnLeft** _(required)_  | **btn-left**  | Define o texto do botão esquerdo, se existir.      | **string** | **undefined** |
| **btnRight** _(required)_ | **btn-right** | Define o texto texto do botão direito, se existir. | **string** | **undefined** |

## Eventos

| Event             | Descrição                                          | Tipo     |
| ----------------- | -------------------------------------------------- | ---------|
| **btnLeftClick**  | Emitido quando é feito um click no botão esquerdo. | **void** |
| **btnRightClick** | Emitido quando é feito um click no botão direito.  | **void** |

## Slots

| Slot               | Descrição                                                   |
| ------------------ | ----------------------------------------------------------- |
| **slot="imagem"**  | Posiciona um elemento imagem ou icone a esquerda do banner. |
| **slot="content"** | Defibe o conetudo de texto do banner.                       |

## Usage

```html
<med-banner header="Notification Title" btn-left="cancelar" btn-right="editar dados">
  <img slot="imagem" src="../../imagem.jpg" alt="Imagem">
  <p slot="content">Lorem ipsum dolor, sit amet consectetur adipisicing elit. Ipsam eaque deserunt quidem! Doloribus voluptatem quod porro facere voluptate et. Minima explicabo placeat doloribus voluptatibus magni iusto consequatur perferendis libero, porro cum. Unde, corrupti alias, vitae aliquid iure asperiores ratione optio vel culpa tempora temporibus, molestias labore? Iusto repudiandae voluptatibus ab.</p>
</med-banner>
```
