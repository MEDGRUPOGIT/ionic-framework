# Usage

### Atributos

| Atributo    | Descrição                                    | Type         |
| ----------- | -------------------------------------------- | ------------ |
| **imagens** | Define uma lista de imagens para o enunciado | **String[]** |

### Eventos

| Event                 | Descrição                                 | Type               |
| --------------------- | ----------------------------------------- | ---------- |
| **medGalleryRequest** | Emitido quando é feito um click na imagem | **String** |
