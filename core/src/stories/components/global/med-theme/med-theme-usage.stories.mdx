# Tema

## Propriedades

| Propriedade   | Atributo      | Descrição                                   | Tipo                                                                                                                | Default       |
| ------------- | ------------- | ------------------------------------------- | --------------------------------------------------| ------------- |
| **dsName**    | **ds-name**   | Define a variação do componente.            | **"theme-gold" \| "theme-recursos" \| undefined** | **undefined** |
| **nome**      | **nome**      | Define a cor do componente.                 | **string \| undefined**                           | **undefined** |

## Default

```
<med-theme ds-name=${dsName} nome="nome"></med-theme>
```

## Theme gold

```
<med-theme ds-name="theme-gold" nome="Theme Gold"></med-theme>
```
