# Rate Like

## Propriedades

| Propriedade  | Atributo   | Descrição             | Tipo           | Default       |
| ------------ | ---------- | --------------------- | -------------- | ------------- |
| **status**   | **status** | Define o botão ativo. | **RateStatus** | **undefined** |

## Eventos

| Event         | Descrição                                                    | Tipo            |
| ------------- | ------------------------------------------------------------ | --------------- |
| **medChange** | Emitido quando é feito um click no botão de like ou dislike. | **RateStatus**  |

```
<!-- default -->

```
enum RateStatus {

  DISLIKE = 'dislike'
}
