# Rate Bar

**Este componente funciona como wrapper para o componente [med-rate-like](http://desenv.ordomederi.com/med-components/?path=/story/components-global-rate-like--default)**.

## Slots

| Slot                 | Descrição                                                                                                                                 |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **slot="avaliacao"** | Slot para o componente **[med-rate-like](http://desenv.ordomederi.com/med-components/?path=/story/components-global-rate-like--default)** |

## Usage

```
<med-rate-bar>
  Avalie esse vídeo

</med-rate-bar>
```
