# Caption

## Propriedades

| Propriedade  | Atributo     | Descrição                                   | Tipo                                                                                                                   | Default       |
| ------------ | ------------ | ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**  | **ds-color** | Define a cor do componente.                 | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**  | **undefined** |
| **dsSize**   | **ds-size**  | Define a variação de tamanho do componente. | **"xxs"** \| **"xs"** \| **"sm"** \| **"md"** \| **"lg"** \| **"xl"** \| **undefined**                                 | **undefined** |

## CSS Custom Properties

| Name               | Description                                   |
| ------------------ | --------------------------------------------- |
| **--background**   | Define a cor de background do componente.     |
| **--color**        | Define a cor de texto do componente.          |
| **--border-color** | Define a cor da borda do componente.          |
