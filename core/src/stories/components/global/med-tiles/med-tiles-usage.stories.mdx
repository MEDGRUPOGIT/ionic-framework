# Tiles

## Propriedades

| Propriedade  | Atributo    | Descrição                                   | Tipo                                                                                                                 | Default       |
| ------------ | ----------- | ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------------- |
| **color**    | **color**   | Define a cor do componente.                 | **[Color](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)**   \| **undefined** | **undefined** |
| **neutral**  | **neutral** | Define a cor neutra do componente.          | **[Neutral](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)** \| **undefined** | **undefined** |
| **titulo**   | **titulo**  | Define titulo do card                       | **Titulo**                                                                                          \| **undefined** | **undefined** |
| **label**    | **label**   | Define a label de texto do card             | **label**                                                                                           \| **undefined** | **undefined** |
| **badge**    | **badge**   | Define o texto do badge                     | **badge**                                                                                           \| **undefined** | **undefined** |
| **selected** | **selected**| Atribui estilo para item checked            | **selected**                                                                                        \| **undefined** | **undefined** |

## CSS Custom Properties

| Name                              | Description                                   |
| --------------------------------- | --------------------------------------------- |
| **--background**                  | Define a cor de background do componente.     |
| **--color-title**                 | Define a cor de texto do titulo.              |
| **--padding**                     | Define o padding.                             |
| **--border-radius**               | Define o border-radius.                       |
| **--border-background**           | Define a cor da borda.                        |
| **--border-background-selected**  | Define a cor da borda quando selecionado.     |
| **--font-size-title**             | Define o tamanho da fonte do titulo           |
| **--font-size-label**             | Define o tamanho da fonte do texto            |
| **--color-label**                 | Define a cor de texto da label                |

## Default

```
<med-tiles titulo="Titulo" label="Texto da label" badge="Selecionar" selected="false"></med-tiles>
```
