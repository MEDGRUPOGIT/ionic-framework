# Progress Bar

## Propriedades

| Propriedade    | Atributo       | Descrição                                   | Tipo                    | Default        |
| -------------- | -------------- | ------------------------------------------- | ----------------------- | -------------- |
| **value**      | **value**      | Define o valor para progresso da barra.     | **number: > 0 <= 0.1**  | **undefined**  |
| **percentage** | **percentage** | Esconde ou mostra a porcentagem.            | **boolean**             | **undefined**  |
| **dsName**     | **ds-name**    | Define a variação do componente.            | **"secondary"**         | **undefined**  |

## Default

```
<ion-progress-bar value="0.3"></ion-progress-bar>
```

### Percentage

```
<ion-progress-bar value="0.3" percentage="true"></ion-progress-bar>
```

## Variação Minimalist

```
<ion-progress-bar value="0.3" ds-name="minimalist"></ion-progress-bar>
```

### Color

```
<ion-progress-bar ds-name="minimalist" color="brand"></ion-progress-bar>
```
