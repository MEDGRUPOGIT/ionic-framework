# Divider

## Propriedades

| Propriedade            | Atributo     | Descrição                                   | Tipo                                                                                                                 | Default       |
| ---------------------- | ------------ | ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**            | **ds-color** | Define a cor do componente.                 | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**    | **undefined** |
| **text**  _(required)_ | **text**     | Define o texto do componente.               | **"string"**                                                                                                         | **undefined** |


## Default

```
<med-divider text="Montar Provas"></med-divider>

<med-divider ds-color="brand" text="Montar Provas"></med-divider>
```
