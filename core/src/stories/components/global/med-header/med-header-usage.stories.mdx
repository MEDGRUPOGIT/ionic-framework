# Header

## Slots

| Slot                | Descrição                                                                                                                                    |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **slot="navbar"**   | Slot para o componente **[med-navbar](http://desenv.ordomederi.com/templarios/?path=/story/components-global-navbar--default)**.         |
| **slot="toolbar"**  | Slot para o componente **[med-toolbar](http://desenv.ordomederi.com/templarios/?path=/story/components-global-toolbar--default)**.       |
| **slot="progress"** | Slot para o componente **[ion-progress](http://desenv.ordomederi.com/templarios/?path=/story/components-global-progress-bar--default)**. |

## Eventos

| Event           | Descrição                               | Tipo       |
| --------------- | --------------------------------------- | ---------- |
| **med-resize**  | Emitido quando a altura do header muda. | **Number** |

## Usage

### med-navbar

```
<med-header>
  <med-navbar slot="navbar">
    <ion-button ds-name="icon-label" slot="left">
      <ion-icon class="med-icon" name="med-chevron-left"></ion-icon>
      voltar
    </ion-button>

    <span slot="title">Título</span>
    <span slot="subtitle">subtítulo</span>

    <ion-button ds-name="icon-only" slot="right">
      <ion-icon class="med-icon" slot="icon-only" name="med-star-filled"></ion-icon>
    </ion-button>
  </med-navbar>
</med-header>
```
