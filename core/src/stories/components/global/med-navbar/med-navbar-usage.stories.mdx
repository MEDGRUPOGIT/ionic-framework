# Navbar

## Propriedades

| Propriedade  | Atributo    | Descrição                                 | Tipo                                                                                                                 | Default       |
| ------------ | ----------- | ----------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------------- |
| **color**    | **color**   | Define a cor do componente.               | **[Color](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)** \| **undefined**   | **undefined** |
| **neutral**  | **neutral** | Define a cor neutra do componente.        | **[Neutral](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)** \| **undefined** | **undefined** |
| **dsName**   | **ds-name** | Define a variação do componente.          | **"secondary"** \| **"transparent"** \| **undefined**                                                                | **undefined** |

## Slots

| Slot                | Descrição                                         |
| ------------------- | ------------------------------------------------- |
| **slot="left"**     | Posiciona um elemento ao lado esquerdo da navbar. |
| **slot="title"**    | Titulo da navbar.                                 |
| **slot="subtitle"** | Subtitulo da navbar.                              |
| **slot="right"**    | Posiciona um elemento ao lado direito da navbar.  |

## CSS Custom Properties

| Name              | Description                                   |
| ----------------- | --------------------------------------------- |
| **--background**  | Define a cor de background do componente.     |
| **--color**       | Define a cor de texto do componente.          |

## Usado por

 - **[med-offline](../med-offline)**

## Default

```
<med-navbar>
  <ion-button ds-name="icon-label" slot="left">
    <ion-icon class="med-icon" name="med-chevron-left"></ion-icon>
    voltar
  </ion-button>

  <h1 slot="title">header</h1>
  <h2 slot="subtitle">subheader</h2>

  <ion-button ds-name="icon-only" slot="right">
    <ion-icon class="med-icon" slot="icon-only" name="med-star-filled"></ion-icon>
  </ion-button>
</med-navbar>
```
