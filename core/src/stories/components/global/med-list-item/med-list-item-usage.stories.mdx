# List Item

## Propriedades

| Propriedade  | Atributo    | Descrição                                   | Tipo                                                                                                                 | Default       |
| ------------ | ----------- | ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------------- |
| **color**    | **color**   | Define a cor do componente.                 | **[Color](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)**   \| **undefined** | **undefined** |
| **neutral**  | **neutral** | Define a cor neutra do componente.          | **[Neutral](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)** \| **undefined** | **undefined** |
| **titulo**   | **titulo**  | Define o titulo do componente.              | **titulo**                                                                                          \| **undefined** | **undefined** |
| **label**    | **label**   | Define o texto do conteudo do componente.   | **label**                                                                                           \| **undefined** | **undefined** |
| **selected** | **selected**| Define se o componente foi checked          | **selected**                                                                                        \| **undefined** | **undefined** |
| **border**   | **border**  | Define se o componente tem borda redonda    | **border**                                                                                        \| **undefined** | **undefined** |
| **dsSize**   | **'xs'  'sm'  'md'** | Define o tamanho do padding        | **ds-size**                                                                                        \| **undefined** | **undefined** |


## CSS Custom Properties

| Name                  | Description                                       |
| --------------------- | ------------------------------------------------- |
| **--padding**         | Define a padding do componente.                   |
| **--background**      | Define o background do componente.                |
| **--border-radius**   | Define o tamanho do border-radius do componente.  |
| **--font-size-title** | Define o tamanho da fonte do titulo do componente.|
| **--color-title**     | Define a cor do titulo do componente.             |
| **--font-size-label** | Define o tamanho da fonte do texto do componente. |
| **--color-label**     | Define a cor do texto do componente.              |
