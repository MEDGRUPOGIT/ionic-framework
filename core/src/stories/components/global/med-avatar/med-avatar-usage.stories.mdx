# Avatar

## Propriedades

| Propriedade   | Atributo      | Descrição                                   | Tipo                                                                                                                | Default       |
| ------------- | ------------- | ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**   | **ds-color**  | Define a cor do componente.                 | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**   | **undefined** |
| **dsSize**    | **ds-size**   | Define a variação de tamanho do componente. | **"xxs"** \| **"xs"** \| **"sm"** \| **"base"** \| **"md"** \| **"lg"** \| **"xl"** \| **"xxl"** \| **undefined**   | **undefined** |
| **image**     | **image**     | Define a imagem do componente.              | **string** \| **undefined**                                                                                         | **undefined** |
| **letter**    | **letter**    | Define a imagem do componente.              | **string** \| **undefined**                                                                                         | **undefined** |
