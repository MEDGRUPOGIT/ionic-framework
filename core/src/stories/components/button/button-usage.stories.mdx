# Button

Documentação auxiliar ion-button: **https://ionicframework.com/docs/api/button**

## primary

### Propriedades

| Propriedade  | Atributo     | Descrição                          | Tipo                              | Default       |
| ------------ | ------------ | ---------------------------------- | --------------------------------- | ------------- |
| **color**    | **color**    | Define a cor do botão.             | **[Color](http://desenv.ordomederi.com/med-components/?path=/story/enums-enums--page#medcolors)**  | **undefined** |
| **dsName**   | **ds-name**  | Define o botão.                    | **"primary"**                     | **"primary"** |
| **expand**   | **expand**   | Define o tamanho do botão.         | **"full"** \| **"block"**         | **undefined** |
| **disabled** | **disabled** | Define o estado disabled do botão. | **boolean**                       | **undefined** |
| **dsSize**   | **ds-size**  | Define o tamanho do botão.         | **"xxs"** \| **"xs"** \| **"sm"** | **undefined** |

```html
<!-- primary -->
<ion-button ds-name="primary">ion-button</ion-button>

<!-- disabled -->
<ion-button ds-name="primary" disabled>ion-button</ion-button>

<!-- expand -->
<ion-button ds-name="primary" expand="full">ion-button</ion-button>
<ion-button ds-name="primary" expand="block">ion-button</ion-button>

<!-- size -->
<ion-button ds-name="primary" ds-size="xxs">ion-button</ion-button>
```

## icon-only

### Propriedades

| Propriedade  | Atributo     | Descrição                          | Tipo                                         | Default         |
| ------------ | ------------ | ---------------------------------- | -------------------------------------------- | --------------- |
| **dsName**   | **ds-name**  | Define o botão.                    | **"icon-only"**                              | **"icon-only"** |
| **dsSize**   | **ds-size**  | Define os tamanhos do botão.       | **"xs"** \| **"sm"** \| **"md"** \| **"lg"** | **undefined**   |
| **disabled** | **disabled** | Define o estado disabled do botão. | **boolean**                                  | **undefined**   |

```html
<!-- icon-only -->
<ion-button ds-name="icon-only">
  <ion-icon slot="icon-only" name="med-chevron-left"></ion-icon>
</ion-button>

<!-- disabled -->
<ion-button ds-name="icon-only" disabled>
  <ion-icon slot="icon-only" name="med-chevron-left"></ion-icon>
</ion-button>

<!-- ds-size -->
<ion-button ds-name="icon-only" ds-size="md">
  <ion-icon slot="icon-only" name="med-chevron-left"></ion-icon>
</ion-button>
```

## icon-label

### Propriedades

| Propriedade  | Atributo     | Descrição                          | Tipo             | Default          |
| ------------ | ------------ | ---------------------------------- | ---------------- | ---------------- |
| **dsName**   | **ds-name**  | Define o botão.                    | **"icon-label"** | **"icon-label"** |
| **disabled** | **disabled** | Define o estado disabled do botão. | **boolean**      | **undefined**    |

### Slots

| Slot              | Descrição                                 |
| ----------------- | ----------------------------------------- |
| **slot="start"**  | Posiciona um elemento no começo do botão. |
| **slot="end"**    | Posiciona um elemento no final do botão.  |

```html
<!-- slot="start" -->
<ion-button ds-name="icon-label">
 <ion-icon name="med-arrow-left-circle" slot="start"></ion-icon>
  voltar
</ion-button>

<!-- slot="end" -->
<ion-button ds-name="icon-label">
  próxima
  <ion-icon name="med-arrow-right-circle" slot="end"></ion-icon>
</ion-button>

<!-- disabled -->
<ion-button ds-name="icon-label" disabled>
  próxima
  <ion-icon name="med-chevron-left" slot="end"></ion-icon>
</ion-button>
```
