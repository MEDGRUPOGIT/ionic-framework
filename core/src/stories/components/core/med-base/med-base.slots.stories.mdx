## Slots

| Slot       | Description                                      |
| ---------- | ------------------------------------------------ |
|            | Slot default.                                    |
| `"end"`    | Conteúdo é posicionado a direita do componente.  |
| `"middle"` | Conteúdo é posicionado ao centro do componente.  |
| `"start"`  | Conteúdo é posicionado a esquerda do componente. |
