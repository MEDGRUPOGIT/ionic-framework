# Base

## Propriedades

| Propriedade  | Atributo      | Descrição                                              | Tipo                                                                                                                   | Default       |
| ------------ | ------------- | ------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**  | **ds-color**  | Define a cor do componente.                            | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**      | **undefined** |
| **radius**   | **radius**    | Define a variação de borde-radius do componente.       | **s02** \| **s04** \| **s08** \| **undefined**                                                                         | **undefined** |
| **spacingV** | **spacing-v** | Define a variação de padding vertical do componente.   | **s00** \| **s02** \| **s04** \| **s08** \| **s12** \| **s16** \| **s24** \| **undefined**                             | **undefined** |
| **spacingH** | **spacing-h** | Define a variação de padding horizontal do componente. | **s00** \| **s02** \| **s04** \| **s08** \| **s12** \| **s16** \| **s24** \| **undefined**                             | **undefined** |

## Slots

| Slot       | Descrição                                         |
| ---------- | ------------------------------------------------- |
| **start**  | Conteúdo é posicionado a esquerda do componente.  |
| **middle** | Conteúdo é posicionado ao centro do componente.   |
| **right**  | Conteúdo é posicionado a direita do componente.   |
