# Chart Radial Label

## Propriedades

| Propriedade | Atributo    | Descrição                                   | Tipo                      | Default       |
| ----------- | ----------- | ------------------------------------------- | ------------------------- | ------------- |
| **total**   | **total**   | Define o valor do total de questões.        | **number**                | **0**         |
| **dsSize**  | **ds-size** | Define a variação de tamanho do componente. | **"lg"** \| **undefined** | **undefined** |

## Interfaces

```
export interface MedChartRadiaItem {
  cor: string;
  label: string;
  quantia: number;
  ignoreBarra: boolean;
};
```
