# Rate Bar

**Este componente funciona como wrapper para o componente [med-rate-like](http://desenv.ordomederi.com/templarios/?path=/story/components-global-rate-like--default)**.

## Propriedades

| Propriedade  | Atributo     | Descrição                                   | Tipo                                                                                                                   | Default       |
| ------------ | ------------ | ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**  | **ds-color** | Define a cor do componente.                 | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**  | **undefined** |

## Slots

| Slot                 | Descrição                                                                                                                                 |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **slot="avaliacao"** | Slot para o componente **[med-rate-like](http://desenv.ordomederi.com/templarios/?path=/story/components-global-rate-like--default)** |

## Usage

```
<med-rate-bar>
  Avalie esse vídeo
  <med-rate-like slot="avaliacao"></med-rate-like>
</med-rate-bar>
```
