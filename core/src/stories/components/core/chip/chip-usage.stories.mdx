# Chip

Documentação auxiliar ion-chip: **https://ionicframework.com/docs/api/chip**.

## Propriedades

| Propriedade  | Atributo     | Descrição                                   | Tipo                                                                                                                 | Default       |
| ------------ | ------------ | ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**  | **ds-color** | Define a cor do componente.                 | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**    | **undefined** |
| **dsName**   | **ds-name**  | Define a variação do componente.            | **"secondary"**  \| **undefined**                                                                                    | **undefined** |
| **disabled** | **disabled** | Define o estado disabled do componente.     | **boolean**                                                                                                          | **undefined** |

## Default

```
<ion-chip>
  <ion-label>UNIFESP 2020</ion-label>
</ion-chip>

<ion-chip>
  <ion-icon class="med-icon" name="med-arrow-left-circle"></ion-icon>
  <ion-label>UNIFESP 2020</ion-label>
</ion-chip>

<ion-chip>
  <ion-icon class="med-icon" name="med-arrow-left-circle"></ion-icon>
  <ion-label>UNIFESP 2020</ion-label>
  <ion-icon class="med-icon" name="med-arrow-right-circle"></ion-icon>
</ion-chip>
```
