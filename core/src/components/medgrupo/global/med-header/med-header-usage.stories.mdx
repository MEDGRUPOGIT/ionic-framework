# Usage

### Slots

| Slot                | Descrição                                                                                                                                   |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| **slot="navbar"**   | Slot para o componente **[med-navbar](http://desenv.ordomederi.com/med-components/?path=/story/components-global-navbar--default)**         |
| **slot="toolbar"**  | Slot para o componente **[med-toolbar](http://desenv.ordomederi.com/med-components/?path=/story/components-global-toolbar--default)**       |
| **slot="progress"** | Slot para o componente **[ion-progress](http://desenv.ordomederi.com/med-components/?path=/story/components-global-progress-bar--default)** |

### Eventos

| Event           | Descrição                              | Type       |
| --------------- | -------------------------------------- | ---------- |
| **med-resize**  | Emitido quando a altura do header muda | **Number** |

## Exemplo com med-navbar apenas

```html
<med-header>
  <med-navbar slot="navbar">
    <ion-button ds-name="icon-label" slot="left">
      <ion-icon name="med-chevron-left"></ion-icon>
      voltar
    </ion-button>

    <span slot="title">Título</span>
    <span slot="subtitle">subtítulo</span>

    <ion-button ds-name="icon-only" slot="right">
      <ion-icon slot="icon-only" name="med-star-filled"></ion-icon>
    </ion-button>
  </med-navbar>
</med-header>
```

## Exemplo com med-navbar e med-toolbar

```html
<med-header>
  <med-navbar slot="navbar">
    <ion-button ds-name="icon-label" slot="left">
      <ion-icon name="med-chevron-left"></ion-icon>
      voltar
    </ion-button>

    <span slot="title">Título</span>
    <span slot="subtitle">subtítulo</span>

    <ion-button ds-name="icon-only" slot="right">
      <ion-icon slot="icon-only" name="med-star-filled"></ion-icon>
    </ion-button>
  </med-navbar>

  <med-toolbar slot="toolbar">
    <ion-button ds-name="icon-label" disabled slot="start">
      <ion-icon name="med-arrow-left-circle" slot="start"></ion-icon>
      anterior
    </ion-button>
    <ul>
      <li>
        <ion-button ds-name="icon-only">
          <ion-icon slot="icon-only" name="med-star-outline"></ion-icon>
        </ion-button>
        <ion-button ds-name="icon-only">
          <ion-icon slot="icon-only" name="med-eye-show-line"></ion-icon>
        </ion-button>
        <ion-button ds-name="icon-only">
          <ion-icon slot="icon-only" name="med-notes"></ion-icon>
        </ion-button>
        <ion-button ds-name="icon-only">
          <ion-icon slot="icon-only" name="med-font-size"></ion-icon>
        </ion-button>
      </li>
    </ul>

      próxima
      <ion-icon name="med-arrow-right-circle" slot="end"></ion-icon>
    </ion-button>
  </med-toolbar>
</med-header>
```
