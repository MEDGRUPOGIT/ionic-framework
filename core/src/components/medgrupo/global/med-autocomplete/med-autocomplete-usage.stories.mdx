# AutoComplete

O componente AutoComplete deve ser usado como complemento do componente **ion-input**. O controle do componente deve ser feito pela tela onde esta sendo aplicado.

## Propriedades

| Propriedade  | Atributo   | Descrição                                                           | Tipo                                                                                       | Default       |
| ------------ | ---------- | ------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ | ------------- |
| **color**    | **color**  | Define a cor do badge baseado nos tokens. **⚠️ omitir med-color-** | **[tokens](http://desenv.ordomederi.com/med-components/?path=/story/tokens-cores--cores)** | **undefined** |
| **fill**     | **fill**   | Define o comportamento de fill do badge.                            | **"outline"**                                                                              | **undefined** |
| **dsSize**   | **ds-size**| Define o tamanho do badge.                                          | **"sm" "md" "lg"**                                                                         | **undefined** |
