# Usage

### Atributos

| Atributo                   | Descrição                                                             | Type                         |
| -------------------------- | --------------------------------------------------------------------- | -----------------------------|
| **alternativas**           | Objeto do tipo **MedAlternativaInterface**                            | **MedAlternativasInterface** |
| **keyAlternativa**         | Define a Key alternativa em caso ela for diferente de **Alternativa** | **String**                   |
| **keyEnunciado**           | Define a Key alternativa em caso ela for diferente de **Enunciado**   | **String**                   |
| **keyImagem**              | Define a Key alternativa em caso ela for diferente de **Imagem**      | **String**                   |
| **keyPorcentagem**         | Define a Key alternativa em caso ela for diferente de **Porcentagem** | **String**                   |
| **respostaCorreta**        | Define a resposta correta                                             | **String**                   |
| **mostraResposta**         | Define se o componente vai motrar ou não a resposta correta           | **Boolean**                  |
| **alternativaSelecionada** | Define a alternativa selecionada                                      | **String**                   |

### Eventos

| Event                 | Descrição                                 | Type                         |
| --------------------- | ----------------------------------------- | ---------------------------- |
| **medChange**         | Emitido quando é feito um click no option | **MedAlternativasInterface** |
| **medGalleryRequest** | Emitido quando é feito um click na imagem | **MedAlternativaInterface**  |

### Interfaces

```ts
export interface MedAlternativaInterface {
  Alternativa: string;
  Enunciado: string;
  Imagem: string;
  Porcentagem: number;
}
```
