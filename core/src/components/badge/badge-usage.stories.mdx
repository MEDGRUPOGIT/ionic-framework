# Usage

| Atributo  | Descrição                                                           | Valores      |
| --------- | ------------------------------------------------------------------- | ------------ |
| **color** | Define a cor do badge baseado nos tokens. **⚠️ omitir med-color-** | **[tokens](http://desenv.ordomederi.com/med-components/?path=/story/tokens-cores--cores)** |
| **fill**  | Define o comportamento de fill do badge.                            | **outline**  |

```html
<!-- default -->
<ion-badge primary>ion-badge</ion-badge>

<!-- color -->
<ion-badge color="brand-primary">ion-badge</ion-badge>

<!-- fill -->
<ion-badge fill="outline">ion-badge</ion-badge>
```
