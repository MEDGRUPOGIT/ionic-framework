# Usage

| Atributo                                                                                                           | Descrição                                   |
| ------------------------------------------------------------------------------------------------------------------ | ------------------------------------------- |
| **[secondary](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-secondary--default)** | Define o esilo secundário para o componente |
| **[tertiary](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-tertiary--default)**   | Define o esilo terciário para o componente  |
| **[icon-only](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-icon-only--default)** | Define o esilo terciário para o componente  |

```html
<ion-button></ion-button>
<ion-button secondary></ion-button>
<ion-button tertiary></ion-button>
<ion-button icon-only></ion-button>
```

## Default

| Atributo                                                                                                            | Descrição                                         |
| ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| **[disabled](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-primary--disabled)**    | Define o comportamento **disabled** o componente  |

| **[full](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-primary--full)**            | Define o estilo **block** para o componente       |
| **[size="sm"](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-primary--size-sm)**    | Define o tamanho **sm** para o componente         |
| **[size="xs"](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-primary--size-xs)**    | Define o estilo **xs** para o componente          |
| **[size="xxs"](http://desenv.ordomederi.com/med-components/?path=/story/components-ion-button-primary--size-xxs)**  | Define o estilo **xxs** para o componente         |

```html
<ion-button></ion-button>
<ion-button disabled></ion-button>
<ion-button block></ion-button>
<ion-button full></ion-button>
<ion-button size="sm"></ion-button>
<ion-button size="xs"></ion-button>

```
