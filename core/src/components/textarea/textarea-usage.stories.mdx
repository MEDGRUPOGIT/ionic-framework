# Usage

Documentaçao complementar:
**https://ionicframework.com/docs/api/textarea**

| Tag                   | Descrição                                                 |
| ----------------------| ----------------------------------------------------------|
| **success**           | Aplica o estilo de sucesso ao ser utilizado no componente |
| **fail**              | Aplica o estilo de falha ao ser utilizado no componente   |


### default

O uso do componente **ion-label** e **ion-icon** é necessário, para estilizar corretamente o componente **ion-textarea**

```html
<ion-label ds-name="stacked">Stacked</ion-label>
<ion-textarea placeholder="Enter Input"></ion-textarea>
<ion-label ds-name="helper">Helper</ion-label>
```

### Success

Para utilizar o input success corretamente, é necessário chamar o componente ion-icon com o svg descrito abaixo(**name="med-check-circle"**). Utilizando a tag **"success"**.


```html
<ion-textarea placeholder="Enter Input" success></ion-textarea>
```

### Fail

O Componente Input Fail, é aplicado ao utilizar as tags **"fail"**


```html
<ion-textarea placeholder="Enter Input" fail></ion-textarea>
<ion-label ds-name="helper" fail>Helper</ion-label>
```
