# Usage

**Componente range pode ser utilizado para em conjunto com outros componentes, por exemplo componente de popover.**

⚠️ Para utilização do popover ver: https://ionicframework.com/docs/api/range

```html
<ion-range max="5">
  <ion-icon slot="start" size="small" name="med-font-decrease"></ion-icon>
  <ion-icon slot="end" name="med-font-increase"></ion-icon>
</ion-range>
```
