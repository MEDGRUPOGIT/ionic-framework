# Usage

Documentaçao complementar:
**https://ionicframework.com/docs/api/label**

| Tag                   | Descrição                                                                                       |
| ----------------------| ------------------------------------------------------------------------------------------------|
| **ds-name="stacked"** | Deve ser utilizado no componente **ion-label** para aplicar o estilo stacked (acima do input)   |
| **ds-name="helper"**  | Deve ser utilizado no componente **ion-label** para aplicar o estilo helper (abaixo do input)   |

### Label stacked

Usada para complementar o componente **ion-input** e adicionar o estilo stacked.

```html
<ion-label ds-name="stacked">Stacked</ion-label>
<ion-input placeholder="Enter Input"></ion-input>
```

### Label helper

Usada para complementar o componente **ion-input** e adicionar o estilo helper.

```html
<ion-input placeholder="Enter Input"></ion-input>
<ion-label ds-name="helper">Helper</ion-label>
```
