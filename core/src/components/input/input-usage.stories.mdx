# Usage

Documentaçao complementar:
**https://ionicframework.com/docs/api/input**

| Tag                   | Descrição                                                               |
| ----------------------| ------------------------------------------------------------------------|
| **success**           | Aplica o estilo de sucesso ao ser utilizado no componente **ion-input** |
| **fail**              | Aplica o estilo de falha ao ser utilizado no componente **ion-input**   |

### Input default

O uso do componente **ion-label** e **ion-icon** é necessário, para estilizar corretamente o componente **ion-input**

```html
<ion-label ds-name="stacked">Stacked</ion-label>
<ion-input placeholder="Enter Input"></ion-input>
<ion-label ds-name="helper">Helper</ion-label>
```

### Input Success

Para utilizar o input success corretamente, é necessário chamar o componente ion-icon com o svg descrito abaixo(**name="med-check-circle"**). Utilizando a tag **"success"**.


```html
<ion-input placeholder="Enter Input" success>
  <ion-icon name="med-check-circle">med-check-circle</ion-icon>
</ion-input>
```

### Input Fail

O Componente Input Fail, é aplicado ao utilizar as tags **"fail"**


```html
<ion-input placeholder="Enter Input" fail></ion-input>
<ion-label ds-name="helper" fail>Helper</ion-label>
```
