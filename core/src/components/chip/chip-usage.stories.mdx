# Usage

Documentaçao complementar:
**https://ionicframework.com/docs/api/chip**

| Tag                   | Descrição                                                                          |
| ----------------------| -----------------------------------------------------------------------------------|
| **outline**           | Aplica o estilo transparente com borda ao ser utilizado no componente **ion-chip** |
| **color**             | Aplica uma cor ao ser utilizado no componente **ion-chip**                         |

O uso do componente **ion-label** e **ion-icon** é necessário, para estilizar corretamente o componente **ion-chip**

### default
```html
<ion-chip>
  <ion-label >teste 123</ion-label>
</ion-chip>
```

### Chip + Avatar
```html
<ion-chip>
  <ion-icon name="med-check-circle">med-check-circle</ion-icon>
  <ion-label >teste 123</ion-label>
</ion-chip>
```

### Chip + Icon
```html
<ion-chip>
  <ion-label >teste 123</ion-label>
  <ion-icon name="med-check-circle">med-check-circle</ion-icon>
</ion-chip>
```

### Chip + Dual Icon
```html
<ion-chip>
  <ion-icon name="med-check-circle">med-check-circle</ion-icon>
  <ion-label >teste 123</ion-label>
  <ion-icon name="med-check-circle">med-check-circle</ion-icon>
</ion-chip>
```

### Chip Outline
```html
<ion-chip outline>
  <ion-label >teste 123</ion-label>
</ion-chip>
```

### Chip Color
```html
<ion-chip color>
  <ion-label >teste 123</ion-label>
</ion-chip>
```
